1 📌 このサイトでわかること

  • 東証スタンダード市場の特徴と投資環境
  • 財務指標を使った銘柄選定の考え方
  • データ分析を活用した投資戦略の検討

2 📊 東証スタンダード市場とは？

日本の株式市場は大きく3つのセグメントに分かれています。

市場 特徴 投資対象
プライム市場 安定性が高いが、機関投資家主体 配当狙い
スタンダード市場 中小企業が多く、安定性と成長性のバランス 両方を狙える
グロース市場 成長性が高いが、リスクも大きい キャピタルゲイン狙い

スタンダード市場は 「成長性」と「安定性」 を兼ね備えており、個人投資家にとって適した市場ではないかと考えます。

3 🔍 投資分析の視点

3.1 1️⃣ 財務指標を用いた投資判断

  • PBR（株価純資産倍率）：低PBRの割安銘柄を選定
  • PER（株価収益率）：収益力の高い企業を評価
  • 配当利回り：安定的なインカムゲインを狙う
  • キャッシュフロー分析：財務の健全性を確認

3.2 2️⃣ 個人投資家の行動と心理

  • プロスペクト理論 を考慮し、リスク管理を重視
  • 企業の成長性 vs. 安定性のバランス を考えた投資戦略

  • RStudio - Rを用いたデータ可視化
